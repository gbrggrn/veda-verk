## Systems Architechture

---

## Functionality

* Role based access
* Users not logged in can:
  - Schedule vegetable pickup
  - Schedule pizza pickup
  - Book courses
  - Read about the garden
* Admin users can:
  - Post new offers
  - Track inventory, history and re-launch previous offers
